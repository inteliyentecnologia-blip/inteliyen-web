# INTELIYEN - Agencia Tech Premium 🚀

Plataforma web oficial de INTELIYEN, una agencia especializada en Desarrollo Web/App, Sistemas de Gestión y Estrategia Digital. Además, incluye un catálogo de capacitaciones corporativas en Inteligencia Artificial y Metodologías Ágiles.

## 🛠️ Tecnologías Utilizadas
* **Frontend:** React 18 + TypeScript
* **Estilos:** Tailwind CSS v4 + UI Components (shadcn/ui)
* **Enrutamiento:** React Router v7
* **Empaquetador/Motor:** Vite
* **Iconos:** Lucide React
