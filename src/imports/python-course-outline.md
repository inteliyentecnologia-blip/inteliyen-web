🚀 Módulo 1: El Encantador de Serpientes (Fundamentos)
Objetivo: Configurar el entorno y entender la lógica básica.

Instalación y Setup: VS Code, Python 3.x y el primer "Hola Mundo".

Variables y Tipos de Datos: Strings (texto), Integers (enteros), Floats (decimales) y Booleanos.

Operadores: Matemáticos y de comparación.

Entrada de datos: Usando input() para que el usuario interactúe.

⚠️ Error Común #1: Intentar sumar un texto con un número (Type Errors).

🛠️ Módulo 2: Controlando el Flujo (Lógica)
Objetivo: Que tu programa tome decisiones.

Condicionales (if, elif, else): La base de la toma de decisiones.

Bucles for: Para recorrer listas o rangos.

Bucles while: Para procesos que se repiten hasta que algo cambie.

🛠️ Ejercicio: Crear un juego de "Adivina el número".

⚠️ Error Común #2: El bucle infinito (olvidar actualizar la variable de control).

📦 Módulo 3: Colecciones y Estructuras de Datos
Objetivo: Organizar mucha información de forma eficiente.

Listas y Tuplas: Cómo guardar grupos de datos ordenados.

Diccionarios: Almacenar datos con Clave-Valor (como una agenda).

Sets (Conjuntos): Para datos únicos.

🛠️ Ejercicio: Crear un sistema de gestión de inventario simple.

⚠️ Error Común #3: Intentar acceder a un índice que no existe en una lista.

⚙️ Módulo 4: Funciones y Modularidad
Objetivo: Escribir código limpio y reutilizable.

Definición de funciones (def): Parámetros y retornos (return).

Scope (Alcance): Variables globales vs. locales.

Módulos y Librerías: Cómo importar herramientas externas (import math, import random).

🛠️ Ejercicio: Calculadora científica modular.

🐛 Módulo 5: Depuración y Manejo de Errores
Objetivo: Que el programa no se rompa cuando algo sale mal.

Bloques try / except: Capturar errores sin que el programa muera.

Tipos de excepciones: ValueError, KeyError, IndexError.

⚠️ Error Común #4: Hacer un except genérico que oculta errores reales.

📂 Módulo 6: Trabajando con el Mundo Real
Objetivo: Persistencia de datos.

Lectura y escritura de archivos: Guardar datos en .txt o .csv.

Introducción a JSON: El estándar para enviar datos por internet.

🛠️ Ejercicio Final: Un gestor de tareas que guarde tus pendientes en un archivo para que no se borren al cerrar el programa.
